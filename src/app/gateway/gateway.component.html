Introduction

LoRaWAN Bristol is a messy network, meaning no handoff between gateways, many gateways can overlap, and as such adding to the network is straightforward and reasonably cheap. Each new gateway adds both potential range, but also robustness. The moto is Every Gateway Helps.

Excited? This means that LoRaWAN Bristol wants more gateways and if your interested, then this page provides details on the process of building and registering a new gateway.
